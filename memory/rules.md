# AI Rules

## Coding Standards
- Never break existing API
- Always use TypeScript
- Prefer reusable components
- Ask before major refactors
- Never expose secrets

## Memory Maintenance (Mandatory)
- **Proactive Reading**: Always read `/memory` files before starting work.
- **Continuous Learning**: Every mistake or non-obvious fix MUST be logged in `lessons.md`.
- **State Sync**: Update `tasks.md` and `progress.md` after completing any chunk of work.
- **Decision Integrity**: Log all significant logical changes in `decisions.md`.
